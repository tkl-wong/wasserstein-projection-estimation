# wasserstein-projection-estimation
Code supplement for the paper "Shape-constrained density estimation with Wasserstein projection" by Takeru Matsuda and Ting-Kam Leonard Wong (https://arxiv.org/a/wong_t_2.html).

- quadratic_programming.R: Implementation of non-decreasing density estimation.
- log_concave.R: Implementation of log-concave density estimation.
