# wasserstein-projection-estimation
Code supplement for the paper "Shape-constrained density estimation with Wasserstein projection" by Takeru Matsuda and Ting-Kam Leonard Wong (https://arxiv.org/a/wong_t_2.html). The file "quadratic_programming.R" implements non-decreasing density estimation. The file "log_concave.R" implements log-concave density estimation.
